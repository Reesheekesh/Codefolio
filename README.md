🚀 CodeFolio

A modern developer portfolio builder built using MERN stack.

🔥 Features

- Create developer profiles
- Add skills & projects
- Live portfolio generation
- Clean UI & responsive design

🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Deployment: Render

⚙️ Installation

Clone repo

git clone https://github.com/Reesheekesh/codefolio.git
cd codefolio

Backend setup

cd Backend
npm install

Create ".env" file:

MONGO_URI=your_mongodb_url

Run backend:

node server.js

Frontend setup

cd frontend
npm install
npm start
